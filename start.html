区块元素

大多数html元素被定义为块级元素和内联元素，块级元素在浏览器显示时，通常会以新行来开始和（结束）
例如 h1, p, ul, table
内联元素在显示时通常不会以新行开始
实例： b, td, a, img

div元素， div元素是块级元素，它可以用于组合HTML元素的容器。
div元素与css配合使用可对大的内容块进行设置样式属性。
div属性的另一个用法是用于文档布局。取代table的老式布局方法。

HTML <span> 元素是内联元素，可用作文本的容器
<span> 元素也没有特定的含义。
当与 CSS 一同使用时, < span> 元素可用于为部分文本设置样式属性。
